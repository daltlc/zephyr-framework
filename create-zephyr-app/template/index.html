Accordion

This page is served by the Zephyr MCP server. Every component here can be discovered and controlled by an AI agent using the MCP tools. Try connecting Claude Desktop — it can open this accordion, switch tabs, and trigger the modal without touching a mouse.
The MCP server bridges Claude Desktop to this page via WebSocket. When an agent calls zephyr_act('#my-modal', 'open'), the server forwards it to the browser, which calls Zephyr.agent.act() on the live DOM. No screenshots, no DOM parsing — just typed function calls.
Zephyr ships 14 components: Accordion, Tabs, Carousel, Modal, Dropdown, Toast, Select, Combobox, Date Picker, File Upload, Infinite Scroll, Sortable, and Virtual List. All driven by CSS after page load — zero runtime JavaScript.

Tabs

Zephyr gives you 14 interactive components with zero runtime JS. Every animation, state change, and transition is pure CSS. The framework JS runs once at page load to register custom elements and attach event listeners — after that, CSS takes over.

The Zephyr.agent API lets AI agents discover and control every component on the page. Call getState() to snapshot all components, describe('#id') to deep-inspect one, and act('#id', 'action') to interact — all without DOM knowledge.

Override --z-* CSS custom properties to theme everything. Change --z-transition-duration, --z-border-radius, --z-shadow, and more. Components inherit from these tokens automatically.
